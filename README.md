# Niigata City Wi-Fi Map

新潟市の公開しているNiigata City Wi-Fiのデータを地図で表示します。

https://circled9.github.io/niigata-city-wifi-map/

## Getting Started

### Prerequisities

node.js and npm is required.

### Installing

1. Install dependencies.
```
npm install
```
2. Build
```
npm run start
```
3. Open http://localhost:3000/

## Deployment

適当なWebサーバーにあげてください。
